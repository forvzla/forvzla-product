¿Cómo invitar a ajustar?

Tres opciones

A + B juntas Recomendado

A) Preview inmediata al subir (auto 3:4). B) Icono ✂ sobre la foto, estilo galería. El editor solo abre si lo toca.

Solo icono sobre foto

Menos texto. Descubrimiento depende del ícono (familiar en galerías).

Abrir editor siempre al subir

Máximo control, pero un paso extra obligatorio aunque la foto ya esté bien. Más fricción en registro.

Implementación: al onchange del file → auto-recorte → mostrar preview con ✂ FAB. Tap ✂ → pantalla editor. Listo → actualiza preview. Confirmar → sube blob final.

Flujo propuesto

Camino feliz (80%)

Foto ya vertical → preview inmediata → confirmar (sin abrir editor).
