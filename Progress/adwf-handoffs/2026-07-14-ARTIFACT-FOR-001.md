## What shipped

Una voluntaria registrada puede subir su foto, **ajustar el recorte** si hace falta, y **generar/descargar su carnet** en PDF sin pedir ayuda al equipo. Cuidadoras Caracas es la primera plantilla en producción.

### Changes (epic PR #1)

- Foto de perfil: bucket `voluntario-fotos`, RPCs `preparar_subida_foto_voluntario` / `confirmar_foto_voluntario` / `url_foto_voluntario`
- Cola `carnet_generaciones` + RPCs `solicitar_carnet`, `estado_carnet`, `url_descarga_carnet`, `carnet_disponible`
- Edge Function **`render-carnet`** (pdf-lib, Deno) — desplegada en Supabase
- **Fondo custom:** coordinadora sube PNG vacío 7×5 in (`coord` → pestaña Carnet); bucket `carnet-plantillas`
- PDF superpone: foto + **NOMBRE COMPLETO** + **CÉDULA** + **NÚMERO DE VOLUNTARIA** (etiquetas + valores)
- UI: `registro.html` (foto opcional + CTA carnet al éxito), `carnet.html` + `carnet.js`, `mi-cuenta.html` (menú Mi carnet)
- Rutas clean URLs (`serve.json`, `vercel.json`); fix logout coord → entrar voluntaria

### Changes (post-merge hotfixes)

- **Móvil:** selector de galería sin `capture="user"`; vista previa PDF con **pdf.js** (canvas, sin botón “Open” del navegador)
- **PDF:** campos de texto más abajo para no chocar con logo del PNG de coord
- **Recorte manual (add-on PM):** preview inmediata al subir + icono ✂ sobre miniatura → editor pantalla completa (registro + carnet + regenerar); EXIF orientation; mockup `mockup-carnet-foto-recorte.html`
- **Routing:** fix bucle de redirección Home → Cuidadoras Caracas en dev (`serve.json` + enlace relativo en `index.html`)
